# CONTRIBUTING
## Шаги для создания хороших задач или запросов на включение.
1. **Создание задачи**: Если у вас есть какая-то хорошая идея или вы обнаружили у меня ошибку, создайте новую задачу в разделе "Issues". Напишите как можно подробнее про вашу идею или же про ошибку.
2. **Fork репозитория**: Создайте копию репозитория на своей учетной записи GitHub: bash git clone https://github.com/ваш пользователь/имя репозитория.git
3. **Создание ветки**: Создайте новую ветку для вашей задачи: bash git checkout -b имя ветки
4. **Внесите изменения**: Внесите необходимые изменения в код.
5. **Закоммитьте изменения**: Используйте команду: git add .(чтобы добавить все файлы), а затем git commit -m "Краткое описание изменений"
6. **Отправьте изменения на GitHub**: Отправьте вашу ветку на GitHub: git push
7. **Создайте Pull Request**: Перейдите на страницу вашего форка на GitHub и нажмите на кнопку "Compare & pull request". Опишите свои изменения и создайте запрос.
